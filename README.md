## Tools

Each tool has its own static page for SEO and internal linking. When a new tool is added to the site, add a new English usage guide in this section and include the tool page link.

### Image Tools

#### Image Compressor

Use Image Compressor to reduce JPG, PNG, or WebP file size directly in your browser. Choose an image from your device, adjust the quality setting to balance size and visual clarity, then preview and download the optimized image.

Page: [https://johnshinetools.com/image-compressor/](https://johnshinetools.com/image-compressor/)

#### Image Converter

Use Image Converter to change an image between JPG, PNG, and WebP formats. Select an image, choose the output format, set quality when the format supports it, and download the converted file.

Page: [https://johnshinetools.com/image-converter/](https://johnshinetools.com/image-converter/)

#### Image Resizer

Use Image Resizer to resize JPG, PNG, or WebP images by width, height, or percentage. Upload an image, review the original dimensions and file size, lock the aspect ratio if needed, choose the output format and quality, then preview and download the resized image.

Page: [https://johnshinetools.com/image-resizer/](https://johnshinetools.com/image-resizer/)

### File Tools

#### PDF Merge

Use PDF Merge to combine multiple PDF files into one document. Choose two or more PDF files, keep them in the order selected by your browser, then merge and download the combined PDF.

Page: [https://johnshinetools.com/pdf-merge/](https://johnshinetools.com/pdf-merge/)

#### PDF Split

Use PDF Split to extract selected pages from a PDF. Upload a PDF, enter page numbers or ranges such as `1-3, 5, 8-10`, then generate and download a new PDF containing only those pages.

Page: [https://johnshinetools.com/pdf-split/](https://johnshinetools.com/pdf-split/)

#### PDF Compressor

Use PDF Compressor to create a cleaner and smaller PDF copy where possible. Upload a PDF, run the compressor to rebuild and optimize the PDF structure, then download the optimized file and compare the result.

Page: [https://johnshinetools.com/pdf-compressor/](https://johnshinetools.com/pdf-compressor/)

#### Word Counter

Use Word Counter to measure writing length and reading metrics. Paste or type text into the input box, then review word count, character count, sentence count, and estimated reading time.

Page: [https://johnshinetools.com/word-counter/](https://johnshinetools.com/word-counter/)

### Text Tools

Page: [https://johnshinetools.com/case-converter/](https://johnshinetools.com/case-converter/)

#### Character Counter

Use Character Counter to measure characters, bytes, lines, and paragraphs. Paste or type text and use the live counts for meta descriptions, ads, bios, forms, messages, and platform limits.

Page: [https://johnshinetools.com/character-counter/](https://johnshinetools.com/character-counter/)

#### Unix Timestamp Converter

Use Unix Timestamp Converter to convert between Unix seconds, Unix milliseconds, local date and time, and UTC time. Enter a timestamp or date, convert it, or use the current time as a quick starting point.

Page: [https://johnshinetools.com/unix-timestamp/](https://johnshinetools.com/unix-timestamp/)

#### Encoding Converter

Use Encoding Converter to encode and decode common web formats. Paste text, choose Unicode, UTF-8, ASCII/NATIVE, Base64, URL, HTML entity, or Hex conversion, then run the conversion and copy the output.
